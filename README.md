## 🧩 Engineering Belief / 工程理念

I believe the next generation of software engineering is not only about writing code faster,  
but about designing better **human-agent collaboration loops**.

我相信下一代软件工程的核心不只是“更快地写代码”，而是设计更好的 **人类 × Agent 协作系统**。

Good agents need:

- clear goals / 清晰目标
- rich context / 充分上下文
- reliable tools / 可靠工具
- verifiable feedback / 可验证反馈
- repeatable loops / 可复用闭环

```txt
GOAL  →  CONTEXT  →  TOOLING  →  EXECUTION  →  VERIFICATION  →  LOOP
目标      上下文      工具化        执行           验证             闭环
```
